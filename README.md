# dhc-vs-ahnaf
2 players DHC vs Ahnaf sword fighting game. Who will win?

![](https://cdn.discordapp.com/attachments/873441703330185250/975006251987652659/byiCpv.gif)

# How to Play
## DHC
* **W**: Jump
* **A**: Left
* **D**: Right
* **Spacebar**: Attack
## Ahnaf
* **Arrow Up**: Jump
* **Arrow Left**: Left
* **Arrow Right**: Right
* **Arrow Down**: Attack
